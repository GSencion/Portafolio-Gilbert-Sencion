Herramientas que aprenderás a usar

HTML5 es un lenguaje de marcado que deriva del SGML diseñado para estructurar textos y relacionarlos en forma de hipertexto. Gracias a Internet y los navegadores web, se ha convertido en uno de los formatos más populares que existen para la construcción de documentos para la web.

CSS3 es un lenguaje de hojas de estilo (CSS, Cascading Style Sheets o lenguaje de hojas de estilo) utilizado para describir la semántica de presentación (el aspecto y formato) de un documento escrito en un lenguaje de marcas. Su aplicación más común es diseñar páginas web escritas en HTML i XHTML.

JavaScript es un lenguaje script basado en objetos implementado originariamente por Netscape Communications Corporation, y que derivó en el estándar ECMAScript. Es conocido sobre todo por su uso en páginas web, pero también se utiliza en otras aplicaciones.

Bootstrap es un framework de desarrollo web gratuito y de código abierto. Está diseñado para facilitar el proceso de desarrollo de los sitios web responsive y orientados a los dispositivos móviles, proporcionando una colección de sintaxis para diseños de plantillas.

PHP es el lenguaje de programación del lado del servidor más usado en la web. Con PHP puedes realizar el backend de aplicaciones web de una manera accesible, sencilla y directa.

Con PHP puedes desarrollar sitios y aplicaciones de todo tipo. PHP es el motor además de los CMS más populares, como WordPress, Joomla, Drupal o Magento, entre muchos otros.

WordPress es un sistema de gestión de contenidos web (CMS o content management system), que en pocas palabras es un sistema para publicar contenido en la web de forma sencilla.

Salidas Profesionales

Maquetador/a Web

Están especializados en traducir el diseño gráfico a HTML5 y a lenguaje de estilos CSS con el objetivo de que los navegadores web interpreten todos los elementos y muestran al usuario el diseño propuesto por el diseñador gráfico.

Desarrollador/a web

Son programadores especializados, o dedicados de forma específica, a desarrollar aplicaciones de la World Wide Web o aplicaciones distribuidas en red que se ejecutan intermediando HTTP desde un servidor web a un navegador web.

Diseñador/a Web

Combinan los cimientos estéticos y artísticos con la tecnología, entendimiento como mera herramienta para comunicar ideas a través del diseño de las páginas impresas y de las pantallas de dispositivos electrónicos de todos los aparatos.
